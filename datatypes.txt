Datatypes can be classified to two
    > Primitive types
    > Reference types

Primitive types
    - null
    - undefined
    - nan
    - number
    - string
    - boolean
    - symbol
    - bigint

Reference/Complex types
    - Object
    - Array

Numbers:
    1. Regular integar
    2. Octal number
    3. Hexadecimal number
    4. Floating point number
    5. Biginteger number

Octal number: (0-7)
let num = 071;
console.log(num);//57

Hexadecimal number: (0-9) & (a-f) and it starts with 0x or OX
let num = 0x1a;
console.log(num);//26

Binary Literals:
To parse binary numbers, use the parseint function.
let e = parseInt('111',2);
console.log(e); // 7
